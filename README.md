# twitch-bot

---

### Установка Windows

    python -m venv env
    .\env\Scripts\Activate.ps1
    python.exe -m pip install --upgrade pip
    pip install -r requirements.txt

### Установка Linux

    python -m venv env
    source \env\bin\activate
    python.exe -m pip install --upgrade pip
    pip install -r requirements.txt
